# Client Case Studies

## Case Study — Emblazer.ai
AI agent development platform built from scratch. Designed to interactively understand user preferences before delivering results. Complete product including AI agent orchestration, user preference mapping, full-stack web application.

## Case Study — Huma
Remote patient monitoring platform for modern healthcare. Helps doctors monitor patient records remotely, run virtual hospital wards, support medical research, develop new diagnostic methods using advanced technology.

## Case Study — Mozilla Foundation
Simple, secure website to encourage support for open-source innovation, trustworthy AI, and community-driven digital change. Goal: communicate impact clearly and inspire greater support.
